# WeTravel

WeTravel is a cross-platform social media travel app built with Expo and React Native. It allows users to share travel posts and explore content from other users.

## Setup and Run

Clone the repository:

```bash
git clone <repository-url>

Move into the project and app directory:
cd WeTravel
cd WeTravelApp

Install the dependencies:
npm install

Run the app in a browser:
npm run web

Run the app on iOS (macOS only):
npm run ios

Run the app on Android
npm run android

Start the Expo development server:
npx expo start
